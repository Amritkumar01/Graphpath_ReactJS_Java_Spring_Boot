# GraphPath (Graph Creation + Finding Shortest Path)

A web application for creating and storing graphs and finding the shortest path with an intuitive interface and efficient algorithms.

## Features
- Interactive graph creation and visualization.
- Compute the shortest path between nodes using Dijkstra’s algorithm.
- Display all possible paths between nodes using DFS.
- Intuitive interface for adding nodes, edges, and distances.
- Real-time updates and graph storage using a MySQL database.

## Technologies Used
- **Front-end:** React.js, D3.js for graph visualization
- **Back-end:** Java Spring Boot
- **Database:** MySQL
- **Algorithms:** Dijkstra’s Algorithm (shortest path), DFS (all paths), Manhattan distance

## Key Highlights
- Designed an interactive platform using React.js for the front-end and Java Spring Boot with MySQL for the back-end, enabling users to build, visualize, and analyze graphs.
- Implemented **Dijkstra’s algorithm** for optimal shortest path computation and **Manhattan distance** for precise node distance calculation.
- Developed **RESTful APIs** using POST and GET requests in Java Spring Boot to store, retrieve, and update graph data in real time, ensuring seamless front-end and back-end communication.
